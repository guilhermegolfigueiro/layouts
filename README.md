# Layouts — Repositório de HTML e CSS

Este repositório contém uma coleção de **layouts estáticos desenvolvidos com HTML e CSS**.  
O objetivo é praticar estruturação de páginas, organização de layouts e conceitos básicos de front-end.

Os arquivos incluem páginas de exemplo (`index.html`, `pag1.html`, `pag2.html`, etc.) e um arquivo de estilos (`style.css`) responsável pelo visual.

## Estrutura do Repositório

A organização do projeto segue o padrão abaixo:

/
├── index.html  
├── pag1.html  
├── pag2.html  
├── pag3.html  
├── style.css  

## Tecnologias Utilizadas

- **HTML5** — Estrutura das páginas  
- **CSS3** — Estilização e layout

Este repositório foi feito para:
